1. need to install pyglet, pygame, kivy
2. need AVBin to decode mp3 music, the installation package in this folder already.
3. you can directly install client.apk in your android cell phone for testing. And, run the pygletlucky.py in server side.
4. if you have multi screens, it will automatically selected the last one. (for example, you havee a stage screen, and local screen, the stage screen will be selected)
5. within code folder:
	run init.py to initial the program data.
	run results.py to check the result whenever you want to review.

1. 要安装 pyglet, kivy, pygame
2. 要安装 AVBIN解码MP3,这个文件已经在压缩文件里了.
3. 可以直接安装client.apk在安卓手机上运行, 电脑服务端运行 pygletserver.py. 手机client.apk上必须设置好电脑服务端的ip.
4. 如果你电脑有多个屏幕, 程序会自动选择最后一个屏幕. (比如你电脑接了一个舞台屏幕,那么程序会自动选择舞台屏幕进行投放)
5. 在code文件夹里:
	运行 init.py 初始化程序的一些数据.
	运行 results.py可以查看抽奖结果, 这个程序可以在你任何时候想看结果的时候都可以运行.
